Descripción: El algoritmo Bubble Sort funciona comparando los numeros que estan al lado, osea el primer elemento con el segundo, si es que el primero es mayor que el de la derecha, su posicion se intercambia y asi hasta cuando el numero sea menor, luego vuelve al inicio y repite el proceso hasta que la lista queda ordenada.

Complejidad computacional:
¿Cuál es la complejidad Big O de su solución?: La complejidad Big O es O(n²)

¿Por qué tiene esa complejidad? (Justifique en base a los loops utilizados):  Porque usa un for dentro de un for entonces si aumenta un elemento necesariamente se debería repetir 2 veces el proceso

¿Qué pasa con el tiempo de ejecución si la lista tiene 10 elementos vs 100 elementos?: Si la lista tiene 10 elementos el programa aún va a ser rápido pero a medida que se vayan aumentando como con 100 elementos el tiempo incrementará porque se tienen que hacer muchas más comparaciones ya que es cuadrática
